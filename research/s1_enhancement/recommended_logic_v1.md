# S1強化ロジック v1（2026-05-24）

## 推奨結論

15年データ（2011-01-04〜2026-05-22）で、S1をそのまま毎日運用するより、以下のフィルタを入れる方が数値上は強い。

```text
前日終値と今日始値の差（ギャップ絶対値）が400円未満なら取引しない。
400円以上ギャップした日だけ、従来S1の方向で取引する。
NYダウが同方向なら従来通り2倍、逆方向なら1倍。
引けで必ず決済。
```

## 推奨運用フェーズ

### Phase 1: 少額・初心者・本番初期

```text
400円未満無視 + S1標準枚数
```

理由:
- 15年で負け年0
- 最大DDが従来S1の約1/5
- 勝率70%超
- 20pt不利約定でも成績が崩れにくい

### Phase 2: 利益を取りに行くバランス型

```text
300円未満無視 + S1標準枚数
```

理由:
- 利益は400円フィルタより増える
- 勝率65%台
- DDは従来S1より大幅に小さい

### Phase 3: 攻め運用

```text
100円未満無視 + S1標準枚数
```

理由:
- 15年利益は従来S1を上回る
- ただしDDとスリッページ耐性は悪化するため、初期運用には不向き

## 懸念

- 400円フィルタは取引数が少ない。2011〜2014はほぼ取引が出ないため、低ボラ局面では機会損失が大きい。
- バックテストは日足ベース。寄り付き後に判定して始値で約定する前提は甘いので、スリッページ込みで判断する。
- 本番前に1分足/5分足で、寄り付き後1〜3分エントリーを検証する必要がある。

## 現時点の採用案

```text
採用: 400円未満無視 + S1標準枚数
準採用: 300円未満無視 + S1標準枚数
保留: 100円未満無視 + S1標準枚数
不採用: 従来S1を毎日そのまま本番初期に使う
```
